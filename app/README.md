# ⚡ Electronic
A Cross Platform and Open Source Electronics Design Automation Suite

## 👾 Os 
- Windows
- MacOS
- Linux

## ⚙️ Configure
- Install [Qt6](https://doc.qt.io/qt-6)
- Install [CMake](https://cmake.org)
- Install [Ninja](https://ninja-build.org)

## 🔨 Build
```
qt-cmake -G Ninja -S . -B build
cd build
ninja
```

## 🚀 Run 
```
./bin/Electronic
```

## Api
- [Grpc](../api)
